Project structure:

  • src/main/java - Java source code
    • controllers - application controllers - write your actions here
    • framework - framework classes - do not change that unless you really need that
    • services - application services - write reusable code for controllers here
    • view - classes used mainly by view - like Helpers - write reusable code for view and controllers here
  • src/main/resources - Groovy templates and other hidden from outside world resources. Every template is placed inside the directory with the name of controller
  • src/main/webapp/resources - static resources available directly from the browser (publicly available)
  • src/main/webapp/WEB-INF - directory with configuration
  • target/generated - contains groovy source files generated from templates
