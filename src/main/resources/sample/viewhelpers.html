This page shows usage of helper methods on the view layer. Helper methods are Java static public methods defined in framework.GlobalHelpers and view.Helpers (user defined) to be used in views and controllers. Check the source code of classes framework.GlobalHelpers and view.Helpers for details. Here are some examples of how to use them in the templates:

Some of framework.GlobalHelpers:

link - generates HTML link (a href) link("index", "Go to index using link()") <%= link("index", "Go to index using link()") %>
action - generates url for action <a href="<%= action('index') %>">Go to index using action()</a> Go to index using action()
resource - generates url for resource <script src='<%= resource("jquery15.js") %>'></script> <!-- see the source code of this page -->
parse - include given template and pass parameters to it <%= parse("_somePartial", ['me':'helpers.html']) %> <%= parse("_somePartial", ['me':'helpers.html']) %>
nvl - returns first parameter or default value if first parameter is null Parameter "paramYouDidNotPass" is "<%= nvl(param('paramYouDidNotPass'), 'not given') %>" Parameter "paramYouDidNotPass" is "<%= nvl(param('paramYouDidNotPass'), 'not given') %>"
