This page shows usage of helper methods on the view layer. Helper methods are Java static public methods defined in framework.GlobalHelpers and view.Helpers (user defined) to be used in views and controllers. Here are some examples of how to use them in the templates:

Some of framework.GlobalHelpers:

link("index", "Go to index using link()")
<%= link("index", "Go to index using link()") %>

<a href="<%= action('index') %>">Go to index using action()
Go to index using action()

<script src='<%= resource("jquery15.js") %>'></script> (see the source code of this page)

<%= parse("_somePartial", ['me':'helpers.html']) %>
<%= parse("_somePartial", ['me':'helpers.html']) %>

Parameter "paramYouDidNotPass" is "<%= nvl(param('paramYouDidNotPass'), 'not given') %>"
Parameter "paramYouDidNotPass" is "<%= nvl(param('paramYouDidNotPass'), 'not given') %>"
